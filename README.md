# ArcaeaAssetsUpdater
An api server that provides auto updates of static resources for nonebot-plugin-arcaea.

## How to use ?

```bash
poetry install

poetry run python ArcaeaAssetsUpdater

curl -X POST -H "Authorization: 616.sb" http://127.0.0.1/api/force_update
```
